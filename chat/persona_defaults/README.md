# Persona templates (committed)

Canonical defaults for Scout's persona workspace. The hub loads them from disk at startup (`scout_persona._read_persona_default`).

- **In Git:** Edit these files to change defaults for new installs and **Reset to defaults** in Persona settings.
- **On disk at runtime:** `chat/staging/persona/` is gitignored; seeded on first use.

See root plan / Persona settings in the Ada-SI dashboard for the full file list and bootstrap ritual.
